    media

    @mixin media($breakpoint) { ... }

    Description

    Sets styles based on a certain screen size

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $breakpoint

    The devices breakpoint

    		String— none

    Example

    .phone-screen-styles{
  @include media(phone){
    flex: 1 1 100%;
  }
}

    Throws

    • breakpoint not suported #{$breakpoint}, must be one of: phone || tabletPortrait || tabletLandscape || smallDesktop || largeDesktop

    color

    @function color($color, $tone: default) { ... }

    Description

    Sets a color and tone on a text element or background

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $color

    The brand color to use

    		String— none
    $tone

    The tone of the color to use

    		Stringdefault

    Returns

    String —

    HSL value of the selected color and tone

    Example

    Setting a text color

    .primary-dark-color {
  color: color(primary, dark);
}

    Setting a background color

    .inverse-background-color {

}
